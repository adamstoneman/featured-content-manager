# Featured Content Manager

This README is for development info only, will be removed during plugin build. Public info should go in readme.txt!

## Release
1. Install the build script 'plugin-build' in your path
2. cd to /build
3. $ plugin-build [version number, eg 0.3]
4. If you answer yes to the question the plugin will be pushed to our plugin server (A zip file of the built plugin will be found in the build folder)
5. To make the update live, login to https://plugins.klandestino.se and change the current version number to the current version
