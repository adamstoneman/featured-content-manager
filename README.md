## Description

Featured Content Manager is a plugin highly influenced by (the now dead projekt) wp-featured-content. It lets users create featured items that mirrors posts â€“ then order them and edit their representation inside featured areas.

## Installation
1. Clone this project to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you are ready to update your theme for Featured Content Manager
